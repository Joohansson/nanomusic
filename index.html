How does it work?

Transactions/blocks are recorded live from the Nano currency network. Amounts are presented as floating text (>x = receive, x> = send).
During each melody a new melody of max 64 notes is created by interpreting incoming block hash, type and amount.
For every new block recorded, the melody is extended and old notes are discarded.
If blocks arriving faster than 64 per 8sec interval, they are ignored.
